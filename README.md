# vim
My personal .vimrc plus other VIM plugins that I can't live without. Includes Pathogen, Syntastic, Vim-Airline, and the Molokai theme.


*To install:*

`git clone git://github.com/bekher/vim.git ~/.vim`

`cp ~/.vim/.vimrc ~/.vimrc`
